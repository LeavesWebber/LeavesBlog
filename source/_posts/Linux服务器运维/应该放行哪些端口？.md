# 一些协议的默认端口
## 文件服务

|        服务名        |        端口号        |   协议    |
| :---------------: | :---------------: | :-----: |
|        AFP        |        548        |   TCP   |
| SMB (netbios-ssn) |        139        | TCP/UDP |
| SMB(microsoft-ds) |        445        | TCP/UDP |
|      NETBIOS      |      137,138      |   UDP   |
|   SSH/SFTP/SCP    |        22         |   TCP   |
|      FTP（控制）      |        21         |   TCP   |
|      FTP（数据）      |        20         |   TCP   |
|        NFS        |   111,892,2049    | TCP/UDP |
|       TFTP        |        69         |   UDP   |
|   WebDAV、CalDAV   | 5005、5006 (HTTPS) |   TCP   |
|       iSCSI       |       3260        |   TCP   |
